# Eclipse UOMo

Eclipse UOMo adds Units of Measurement support for IT Systems and Services both to common programming languages like Java and Data Exchange standards like UCUM and formats including UnitsML, SensorML, MathML, general XML or JSON.

The project contains the following bundles:

- [UOMo Core](bundles/org.eclipse.uomo.core)
- [UOMo Business](bundles/org.eclipse.uomo.business)
- [UOMo ICU](bundles/org.eclipse.uomo.icu)
- [UOMo UCUM](bundles/org.eclipse.uomo.ucum)
- [UOMo UI](bundles/org.eclipse.uomo.ui)
- [UOMo Util](bundles/org.eclipse.uomo.util)
- [UOMo XML](bundles/org.eclipse.uomo.xml)

## Requirements
UOMo uses modular unit systems on top of the JSR 385 standardard [Unit-API](https://unitsofmeasurement.github.io/unit-api/) and its Reference Implementation [Indriya](https://unitsofmeasurement.github.io/indriya/).
